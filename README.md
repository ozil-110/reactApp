## 运行demo

### 环境

安装node，mongodb；配置webpack

### 准备

修改src/constant/config.js文件，指定自己的数据库地址，端口，数据库名；node服务器端口，ip；启动mongodb服务


### 开发环境

推荐

```js
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install
```
#### 运行

```js
npm run run-dev-server
npm run dev
```

### 生产环境


```js
npm run start
npm run build
```
